# Modularity Mosaic

A minimal Modularity module plugin for Municipio.

## Features

- Registers a Modularity module: `Mosaic`
- Adds one ACF field: `Rubrik` (`mod_mosaic_rubrik`)
- Renders only the `Rubrik` value in frontend

## Requirements

- PHP 8.0+
- Municipio / Modularity
- Advanced Custom Fields
